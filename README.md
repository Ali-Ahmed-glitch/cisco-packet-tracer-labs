## About This Repository

This repository contains a series of practical networking labs implemented in Cisco Packet Tracer. Each lab builds on the previous one, covering core networking concepts from basic routing all the way to enterprise-grade topologies with firewalls, VLANs, ACLs, and NAT.

Every lab includes:
- A full README with objectives, topology, addressing table, configuration commands, and verification
- Screenshots of topology, CLI output, and ping tests
- The `.pkt` Cisco Packet Tracer project file
